# Reaction test tool on Python
Simple tool to test and track your reaction time
## Main features:
* Portable single-executable build with embedded graphical assets.
* Real-time reaction time tracking and logging.
* Interactive performance chart displaying the last 20 sessions.
* Persistent history storage saved to the user's AppData directory.
* 1000 frame rate cap.
